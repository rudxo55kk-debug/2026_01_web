CSS 박스 모델 & 레이아웃 실습

1. 박스 모델 (content / padding / border / margin)

기본 박스

content + padding + border + margin이 모두 적용됩니다.

작은 박스

padding과 margin이 작고 border도 얇습니다.

큰 박스

padding과 margin, border가 모두 큽니다.

넓은 박스

width가 넓어지고 배경색도 달라집니다.

box-sizing: border-box;를 지정하면 padding과 border가 width에 포함되어 계산되므로, 너비가 예측 가능해집니다.

3. position: static / relative / absolute / fixed

position: static (기본값) - 문서 흐름에 따라 배치됩니다.
position: relative; top/left 값으로 이동하지만 문서 흐름에는 남습니다.
position: absolute; 부모 중 position이 설정된 요소를 기준으로 위치가 정해집니다.
static은 기본 위치에 있고, relative는 문서 흐름에 남지만 위치가 이동합니다. absolute는 문서 흐름에서 벗어나서 부모를 기준으로 위치가 정해집니다.
position: fixed; 화면의 특정 위치에 고정되며 스크롤해도 붙어있습니다.
