CSS Transition & Animation 예제

1. Transition (hover 기반 부드러운 변화)

Transition은 상태 변화를 감지하여 해당 속성을 부드럽게 전환합니다. 주로 :hover, :focus 등에서 사용됩니다.

버튼 색상/크기 전환

버튼에 transition을 적용해, 마우스를 올릴 때 배경색과 위치가 부드럽게 바뀝니다.

박스 스케일 전환

박스에 transform: scale()를 사용하여 hover 상태에서 크기를 확대하고 그림자를 진하게 만듭니다.

텍스트 색상 전환

텍스트에 color 전환을 적용하여 hover 시 색상이 변경되도록 합니다.
