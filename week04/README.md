# 2026년 3월 26일 학습 정리

## 오늘 학습한 내용 (순서대로)

1. 객체 생성과 메서드 활용

- `student.html`을 생성하고, 이름/학번/전공 속성을 가진 객체를 만들었다.
- `prompt`로 사용자 입력을 받아 객체 속성에 저장했다.
- 객체 메서드(`introduce`)로 자기소개 문장을 만들어 `div#result`에 출력했다.

2. 배열 생성, 평균 계산, filter 사용

- `array.basuc.html`을 생성하고 점수 10개 배열을 다뤘다.
- 0~100 범위의 무작위 점수를 생성해 배열에 저장했다.
- 평균을 계산하고, `filter`를 이용해 60점 미만 점수만 추출해 화면에 출력했다.

3. for 반복문으로 재작성

- 배열 생성과 합계 계산 과정을 `Array.from`, `reduce` 대신 `for` 반복문으로 직접 구현했다.
- 반복문 안에서 난수 생성, 배열 저장, 합계 누적을 함께 처리했다.

4. 이미지 전환 기능 구현

- `change_image.html`을 생성하고 버튼 클릭 시 `img`의 `src`가 바뀌도록 구현했다.
- 두 개의 이미지 URL을 준비해 클릭마다 이미지가 전환되게 만들었다.

5. count 기반 홀짝 전환 로직 적용

- `src` 비교 방식 대신 클릭 횟수 `count`를 사용했다.
- `count`가 홀수면 첫 번째 이미지, 짝수면 두 번째 이미지가 출력되도록 수정했다.

6. 코드 주석 추가

* 이미지 전환 코드에 핵심 흐름(배열, 카운트, 이벤트, 홀짝 조건)을 설명하는 주석을 추가해 가독성을 높였다.
